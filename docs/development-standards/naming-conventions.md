# Naming Conventions

## Files

Use:

lowercase-kebab-case

Example:

session-service.ts

cultivar-controller.ts

analytics-engine.ts

---

## Classes

PascalCase

Example:

SessionService

CultivarController

AnalyticsEngine

---

## Variables

camelCase

---

## Constants

UPPER_SNAKE_CASE

---

## Interfaces

PascalCase

Example:

UserSession

CultivarProfile

AnalyticsResult

---

## Database

Tables

snake_case plural

Examples:

users

grow_sessions

cultivars

breeders

session_ratings

Columns

snake_case

Examples:

created_at

updated_at

cultivar_id
